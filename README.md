# CV Interactif — Djiby Sow

## Description
Ce projet est un CV interactif développé en HTML, CSS et JavaScript.
Il présente mon profil, ma formation, mes compétences, mes projets 
et mes expériences professionnelles sous forme de site web moderne 
et responsive.
Le site inclut des fonctionnalités interactives comme le dark mode, 
un menu hamburger mobile, des barres de compétences animées, 
et la validation du formulaire de contact.

## Technologies utilisées
- HTML5 : structure sémantique du CV
- CSS3 : mise en page avec Grid et Flexbox, responsive design, dark mode
- JavaScript (Vanilla JS / ES6) : interactivité complète
- Google Fonts (Poppins) : typographie moderne
- Font Awesome 6 : icônes
- IntersectionObserver API : animation des barres au scroll
- localStorage : mémorisation du thème choisi

## Fonctionnalités
- Layout 2 colonnes (sidebar fixe + contenu principal)
- Mode sombre / clair avec mémorisation (localStorage)
- Menu hamburger responsive sur mobile
- Barres de compétences animées à l'entrée dans l'écran
- Formulaire de contact avec validation JavaScript côté client
- Bouton retour en haut avec apparition au scroll
- Smooth scroll sur les liens de navigation

1. Télécharger ou cloner le dépôt GitHub
2. Ouvrir le dossier mon-cv/
3. Double-cliquer sur index.html pour l'ouvrir dans un navigateur
4. Aucune installation ni serveur requis — fonctionne directement
